# Integrated Geospatial Assessment of Purulia District

This project presents an integrated geospatial assessment of Purulia District, West Bengal using Google Earth Engine and QGIS.

## Study Area
Purulia District, West Bengal, India

## Datasets
- Landsat 8 Collection 2 Level-2
- ESA WorldCover 2021
- GADM Boundary

## Methods
- Land Surface Temperature (LST)
- NDVI
- NDWI
- Land Use/Land Cover (LULC)
- Pearson Correlation

## Software
- Google Earth Engine
- QGIS

## Outputs
- Study Area Map
- LST Map
- NDVI Map
- NDWI Map
- LULC Map
